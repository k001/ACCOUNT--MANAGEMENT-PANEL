The Constellation Admin Skin

This template is intended for large admins: it provides a wide range of CSS styles for displaying data and powerful JS/Php functions. This documentation will guide you into using and customizing this skin, with detailled guides on how to use CSS classes and template functions.

Technologies

The template is built on HTML5 with CSS3 features. It is compatible with Internet Explorer 7+, Firefox 3+, Opera 9+, Safari 3+, Chrome 3+ and any modern browser. It uses CSS shadows and gradients (created here) to reduce required images for backgrounds, thus reducing bandwith use.

It is powered by jQuery, on a non-obtrusive way: plug-ins are written to work even with jQuery.noConflict() on.

Mobile version

The template provides a mobile version, which is compatible with most webkit-based mobile browsers (iPhone, Android...). All satellite pages (login, error, error documents...) are compatible with both standard and mobile templates, by using CSS media queries.
