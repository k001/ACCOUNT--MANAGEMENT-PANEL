Functions overview

Global template functions
The template declares a number of generic functions: fold/expand, collapsible fieldsets, ajax loading effect, enable/disable buttons...
It also create a setup mechanism which allows applying all template effects to dynamically loaded/created content, and a function to easily display messages within a content block.
Mobile template functions
The mobile template comes with a iphone-like multi-level menu.
Tabs
The template can use as many tabs as needed, even nested, and has a mechanism to enable to back/forward buttons and restore tabs on page refresh. Tab content boxes can also be automatically resized to the same height.
Standard template functions
The standard template provides a powerful context-menu and a function to display notification messages, with some options to display/hide close button, hide after a given timeout...
Tips
Display tips on any element of your page, just by adding a class to it!
AccessibleList
This plugin is usefull to split a long list into pages, and shorten it even more by hiding extra results.
SearchField
A simple ajax script will do ajax queries to retrieve search results, and use the accessibleList plugin to display them. Works together with the php search script
Ajax login script
This is a simple ajax login script, compatible with non-js browsers. Use it as a start to build your own script.
Error handling
An full working example of how to use the template error page with errors/exceptions handling.
CSS/JS compression
Reduce your bandwith use by combining and compressing CSS and JS files.
Search function
A simple script for returning search results for the ajax search field
