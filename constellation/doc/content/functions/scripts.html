Demo scripts

  • Required files:
    login.php, _error/error.php

Login

The template provides one example login page, ready to use. In the php version of the template, the file login.php contains a fully working login page exemple, with AJAX credentials checks and fallback if javascript is disabled.

To learn more, take a look at the code, it is well commented and self-explanatory.

Error

Two example pages are included: error message with detailled reporting, and error report sending page. The php version of the template also include a complete error reporting system, ready to use.

Error handling/displaying/reporting is done in 3 files, and one more for the demo:

_errors/error.html/php
The main error page, displays error message, and can also provide detailled error log for developers.
_errors/sendReport.php
An example page for sending error report from the error page. All you need to do is add some code to send e-mail (example provided in comments) or log error data.
_errors/libError.php
Contains functions for the error pages.
error.php
This is an example page on how to get custom php error handlers to use the template error pages.

Note that all pages in folder _errors (except sendReport.html/php) are intended to be loaded either by the HTTP server or by a php script, so relative pathes to CSS/JS/images won't work if you open them directly.

To learn more, take a look at the code, it is well commented and self-explanatory.
