    common.js, jquery.accessibleList.js
  • Available in:
    standard + mobile

About AccessibleList

This plugin is intended for shortening and paginating long lists, so it won't break your page layout or disturb the user. In the template, it is mainly used for the status bar drop-down lists.

Using the plugin

To use the plugin on a list, just call:

$(selector).accessibleList(options);

options is an object with any of the options below.

Available options

moreAfter
Max number of visible lines in each list above 'more' button (0 or false for no masking) (default: 2)
pageSize
Number of visible matches per page (0 for no pagination) (default: 7)
startPage
Number of page displayed on startup (default: 1)
openedOnStart
Tell wether the list should be expanded on startup (ignored if moreAfter is on 0) (default: false)
animate
Enable animation on list expand/page change (default: true)
moreText
Text for the 'more' button (default: 'More')
lessText
Text for the 'less' button (default: 'Less')
after
Callback function after collapsing/expading/changing page. 'this' is the target list (default: false)

You can change the function's defaults at runtime like this:

// Example: change 'more' button text
$.fn.accessibleList.defaults.moreText = 'Want more?';
