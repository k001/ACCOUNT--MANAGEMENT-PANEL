CSS/JS compression

  • Required files:
    mini.php, jsmin.php
  • Available in:
    standard + mobile

About compression

To reduce bandwith use and number of request to the server, the template includes a simple yet efficient system to combine/compress CSS & Javascript files with php. To use it, just call the file mini.php id the js/ or the css/ folder:

Combine JS files

<!--

Example: load of jquery-1.4.2.min.js, common.js, standard.js and jquery.tip.js

-->

<!-- Standard: 4 files, 4 requests and 113 ko -->
<script type="text/javascript" src="js/jquery-1.4.2.min.js"></script>
<script type="text/javascript" src="js/common.js"></script>
<script type="text/javascript" src="js/standard.js"></script>
<script type="text/javascript" src="js/jquery.tip.js"></script>

<!-- Combined: 1 file, 1 request and 29.3 ko (if compression available) -->
<script type="text/javascript" src="js/mini.php?files=jquery-1.4.2.min,common,standard,jquery.tip"></script>

Combine CSS files

<!--

Example: load of reset.css, common.css, form.css and standard.css

-->

<!-- Standard: 4 files, 4 requests and 114 ko -->
<link href="css/reset.css" rel="stylesheet" type="text/css">
<link href="css/common.css" rel="stylesheet" type="text/css">
<link href="css/form.css" rel="stylesheet" type="text/css">
<link href="css/standard.css" rel="stylesheet" type="text/css">

<!-- Combined: 1 file, 1 request and 14.4 ko (if compression available) -->
<link href="css/mini.php?files=reset,common,form,standard" rel="stylesheet" type="text/css">

There can be as many files as needed, just list them separated by a comma. Note that the file extension is not required to reduce string length, but you can leave it on.

Cache

To limit server load, this system uses a cache (one for js and one for css): the combined file is created once, them served from cache folder. If a file is modified, the cache is automatically refreshed.

Cache files are located in the cache folder in both js and css folders. Note that the cache cleaning mechanism only deletes cache files after an hour, so if you make many changes to your css/js files you may end up with many old cache files. Just let the system clear them after a little while, or remove every files if you need to empty folder.

You may delete cache files any time, they will be re-created automatically.

Compression

The system uses files compression to reduce bandwith use even more. To be available, it only requires the php zlib extension (if not activated, it will try to enable it at runtime).

The zlib extension will first check which compression system the user browser can handle, and then uses it to return combined files.
