Tips plugin

  • Required files:
    common.js, jquery.tips.js
  • Available in:
    standard + mobile

About tips

The tip plugin is an easy way to show elements' title attribute or any additional information. It can be used in two ways:

CSS
Just add a specific class to your elements, and the template setup will add tips on them.
Javascript
The template provides several functions to add tips and control them.

Tip bubble content

The tip bubble content is set automatically following this logic:

  • If the element has a title attribute, it is used.
  • If not, the element text will be used. This works even if the text is in children elements (see jQuery text() method).
  • If none, the tip will contain "No tip".

Note that when using javascript, you'll be able to pass a specific content in the function's options.

CSS

To add a tip on an element, add the class with-tip to it:

<!-- example on an image -->
<img src="images/demo/files/file1-mini.png" width="64" height="45" class="with-tip" title="Image title">

If you need to add tips on every element within another element, just add the class with-children-tip to the parent element:

<ul class="tags with-children-tip">
	<li class="tag-time" title="Click to edit tag: Sea">Sea</li>
	<li class="tag-tags" title="Click to edit tag: Fish">Fish</li>
	<li class="tag-user" title="Click to edit tag: Bubble">Bubble</li>
</ul>
  • Sea
  • Fish
  • Bubble

Position

It is also possible to control tag position with classes:

tip-right
Tip on the right
tip-bottom
Tip below
tip-left
Tip on the left
<ul class="simple-list with-icon with-children-tip" style="width:15em">
    <li><a href="#">Tip on top (default)</a></li>
    <li class="tip-right"><a href="#">Tip on the right</a></li>
    <li class="tip-left"><a href="#">Tip on the left</a></li>
    <li class="tip-bottom"><a href="#">Tip below</a></li>
</ul>
  • Tip on top (default)
  • Tip on the right
  • Tip on the left
  • Tip below

Classes on the same principle as with-children-tip:

children-tip-right
Tip on the right for children elements
children-tip-bottom
Tip below for children elements
children-tip-left
Tip on the left for children elements

Javascript

To dynamically add a tip on an element, use the following code:

$(selector).tip(options);

The available options are:

content
Content of the tip (may be HTML) or false for auto detect (default: false)
onHover
Show tip only on hover (default: true)
reverseHover
Hide tip on hover in, show on hover out (default: false)
stickIfCurrent
Enable permanent tip on elements with the class 'current' (default: false)
currentClass
Name of the 'current' class (default: 'current')
offset
Offset of the tip from the element (default: 4)
position
Position of the tip relative to the element (default: 'top')
animationOffset
Offset for the animation (pixels) (default: 4)
delay
Delay before tip shows up on hover (milliseconds) (default: 0)

You can change the function's defaults at runtime like this:

// Example: change delay
$.fn.tip.defaults.delay = 1000;

Other functions

/**
 * Add tip (if not exist) and show it with effect
 * @param object options same as the tip() method
 */
$(selector).showTip(options);

/**
 * Hide then remove tip
 */
$(selector).hideTip();

/**
 * Update tip content
 * @param mixed content any content (text or HTML) for the tip, of false for automatic detection
 */
$(selector).updateTipContent(content);

/**
 * Call this function to refresh tips when using the stickIfCurrent option 
 * and the 'current' element has changed
 */
$(selector).refreshTip();
