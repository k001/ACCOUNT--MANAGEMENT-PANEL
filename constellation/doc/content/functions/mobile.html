Menu

The mobile template has a built-in menu that opens and navigates like an iPhone menu. No additionnal JS is needed, the menu is automatically activated by the mobile template. Here is how the markup should be:

<!-- Main wrapper -->
<div id="menu">
	
	<!-- Menu button -->
	<a href="#">Menu</a>
	
	<!-- Menu content -->
	<ul>
		<li><a href="#">Write</a>
			<ul>
				<li><a href="#">Articles</a></li>
				<li><a href="#">Add article</a></li>
				<li><a href="#">Posts</a></li>
				<li><a href="#">Add post</a></li>
			</ul>
		</li>
		<li><a href="#">Stats</a>
			<ul>
				<li><a href="#">Global stats</a>
					
					<!-- There can be infinite nested levels -->
					<ul>
						<li><a href="#">Views</a></li>
						<li><a href="#">Unique visitors</a></li>
						<li><a href="#">Sources</a></li>
					</ul>
				</li>
				<li><a href="#">Server load</a></li>
				<li><a href="#">Website stats</a></li>
				<li><a href="#">Reports</a></li>
				<li><a href="#">logs</a></li>
				<li><a href="#">Settings</a></li>
			</ul>
		</li>
		<li><a href="#">Users</a>
			<ul>
				<li><a href="#">List</a></li>
				<li><a href="#">Add user</a></li>
				<li><a href="#">Settings</a></li>
			</ul>
		</li>
		<li><a href="#">System settings</a>
			<ul>
				<li><a href="#">General</a></li>
				<li><a href="#">Server config</a></li>
				<li><a href="#">User groups</a></li>
			</ul>
		</li>
		<li><a href="#">My profile</a></li>
		
		<!-- The class red adds a background to highlight some element -->
		<li class="red"><a href="#">Logout</a></li>
	</ul>
	
</div>
