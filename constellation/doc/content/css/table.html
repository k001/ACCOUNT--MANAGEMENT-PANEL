Table

  • Required files:
    common.css, table.css
  • Available in:
    standard + mobile

The file table.css define 2 main classes : table, for table styling, and head, which allows creating a table-header-like element.

Table example

To use the built-in table style, just add the class table to any table. Note that a valid markup (rows should be in a <tbody> element, headers in a <thead>...) is required.

Available elements

Black cell

This is a special cell style, which can be used anywhere in the table. It is primarily intended for loading statuses, with 3 built-in statuses:

<!-- Loading -->
<th class="black-cell"><span class="loading"></span></th>

<!-- Success -->
<th class="black-cell"><span class="success"></span></th>

<!-- Error -->
<th class="black-cell"><span class="error"></span></th>
