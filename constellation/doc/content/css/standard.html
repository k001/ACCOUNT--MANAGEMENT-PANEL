Columns

There are 3 styles of columns : 2, 3 and 4 columns. You can use them wherever needed, but it is best to wrap them in a div with the columns class:

<div class="columns">
	
	<!-- Columns blocks here -->
	
</div>

This block will automatically expand to fit the inner columns, most of which are floating. So there's no need to add a clearing <div> after the columns.

2 columns

<div class="columns">
	
	<!-- Left column -->
	<div class="colx2-left">
		Left column content
	</div>
	
	<!-- Right column -->
	<div class="colx2-left">
		Right column content
	</div>
	
</div>

Content columns

These columns are designed to be used within a content block, and fit all available space:

<div class="content-columns">
	
	<!-- Vertical separator -->
	<div class="content-columns-sep"></div>
	
	<!-- Left column -->
	<div class="content-left"><div class="dark-grey-gradient with-padding" style="height:200px;">
		Left column content
	</div></div>
	
	<!-- Right column -->
	<div class="content-right"><div class="dark-grey-gradient with-padding" style="height:200px;">
		Right column content
	</div></div>

</div>
