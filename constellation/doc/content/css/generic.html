Basic styles

The template declares some basic styles at the beginning of the file common.css, whose name are quite self-explanatory:

.align-left, .align-center, .align-right
Text align classes
.with-margin, .small-margin, .medium-margin, .large-margin, .no-bottom-margin
These classes add or remove bottom margin
.margin-left, .margin-right, .gutter-left, .gutter-right
These classes add margin on right or left sides
.with-padding
This class add a 1em padding
.float-left, .float-right
Standard floating classes
.relative, .absolute
Classes for positioned elements (read more)
.picto
This class is designed for 16px height images displayed inline: it makes them fit the correct line-height
.clearfix
This is a usefull class which allows to adjust a block to its inner floated elements height
To learn more : http://perishablepress.com/press/2008/02/05/lessons-learned-concerning-the-clearfix-css-hack

Message

Messages can use two different node types : <p> or <ul>, depending on if you need to display only one message at a time, or several:

<p class="message"></p>

  • <ul class="message">
  • <li>Message</li>
  • <li>Message</li>
  • </ul>

There are five differents message styles:

<p class="message"></p>

<p class="message warning"></p>

<p class="message error"></p>

<p class="message success"></p>

<p class="message loading"></p>

Close button

You can add a close button to any element: just add the following code, and the template will make it remove its direct parent node when clicked.

	<span class="close-bt"></span>

For instance:

	<p class="message"><span class="close-bt"></span>My message</p>

My message

For <ul> messages:

	<ul class="message">
		<li>My message</li>
		<li>My message</li>
		<li class="close-bt"></li>
	</ul>
  • My message
  • My message

Note that the parent element need to have position:absolute or position:relative. You can add the class relative to it if needed (read more about positioning):

<div class="box dark-grey-gradient relative">
	<span class="close-bt"></span>
	Place box content here
</div>

Progress bar

There's a generic progress bar style, which can be used with the HTML5 element <progress>:

<progress class="progress-bar">
	<span style="width: 68%">68%</span>
</progress>

68%

Note that the <progress> is a block element, so it shouldn't be used inside of inline elements. In this case, use a span instead:

<span class="progress-bar">
	<span style="width: 68%">68%</span>
</span>

There are 3 other colors:

<progress class="progress-bar">
	<span class="green" style="width: 68%">68%</span>
</progress>

68% 

<progress class="progress-bar">
	<span class="orange" style="width: 68%">68%</span>
</progress>

68% 

<progress class="progress-bar">
	<span class="purple" style="width: 68%">68%</span>
</progress>

68%

You can add animated stripes:

<progress class="progress-bar">
	<span class="with-stripes" style="width: 75%"></span>
</progress>
