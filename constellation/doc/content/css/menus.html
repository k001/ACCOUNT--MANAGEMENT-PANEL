Drop-down menu

Available on: standard template

This is a handy standard drop-down menu, with support for infinite nested levels and icons. It works without any JS, but uses a little jQuery to add fading effect and revert sub-menus that would show out of the screen. Here is the required markup for this menu (in a button in this example):

<!-- Note the required class: with-menu -->
<div class="button with-menu">
	Actions
	
	<!-- Here comes the menu -->
	<div class="menu">
	
		<!-- This is the arrow down image -->
		<img src="images/menu-open-arrow.png" width="16" height="16">
		
		<!-- Menu content -->
		<ul>
			<li class="icon_export"><a href="#">Choose folder</a>
				<ul>
					<li class="icon_network"><a href="#">Website</a></li>
					<li class="icon_server"><a href="#">Server root</a></li>
				</ul>
			</li>
			<li class="icon_refresh"><a href="#">Reload</a></li>
			<!-- This is a disabled menu line: -->
			<li class="icon_search">Search</li>
			<li class="sep"></li>
			<li class="icon_terminal"><a href="#">Custom search</a></li>
		</ul>
		<!-- End  of menu content -->
	</div>
	<!-- Menu end -->
</div>
Actions
  • Choose folder
    • Website
    • Server root
  • Reload
  • Search
  • Custom search

The class with-menu is required on the parent element to position the menu and the opening arrow.

If you need the whole element to open the menu, here is another markup:

<!-- Note the required class: menu-opener -->
<div class="button menu-opener">
	Actions
	
	<!-- This is the arrow down image -->
	<div class="menu-arrow">
		<img src="images/menu-open-arrow.png" width="16" height="16">
	</div>
	
	<!-- Menu content -->
	<div class="menu">
		<ul>
			<li class="icon_export"><a href="#">Choose folder</a>
				<ul>
					<li class="icon_network"><a href="#">Website</a></li>
					<li class="icon_server"><a href="#">Server root</a></li>
				</ul>
			</li>
			<li class="icon_refresh"><a href="#">Reload</a></li>
			<!-- This is a disabled menu line: -->
			<li class="icon_search">Search</li>
			<li class="sep"></li>
			<li class="icon_terminal"><a href="#">Custom search</a></li>
		</ul>
	</div>
	<!-- End  of menu content -->
</div>

The drop-down can be used in many elements: main menu (see demo files for example), inline-blocks... If you need to use it in a block title, there's a little bit of extra markup, because while this would work...

<h1 class="with-menu">
	Title
	
	<!-- Here comes the menu -->
	<div class="menu">
		...
	</div>
	<!-- Menu end -->
</h1>

... it is better to use this kind of markup, with a wrapping <div>:

<div class="h1 with-menu">
	<h1>Title</h1>
	
	<!-- Here comes the menu -->
	<div class="menu">
		...
	</div>
	<!-- Menu end -->
</h1>

Note: if the content-block is not wide enough for its menu, this menu may be covered by the next content-block (normal z-index behaviour). To prevent this, you'll need to give it a slightly higher index: add the class upper-index to the content-block to give it a z-index of 100.

<div class="content-block upper-index">
	<div class="menu">
		...
	</div>
</div>

	...
</div>

Mini menu

This is an contextual menu which shows up when its parent element is hovered. Usefull for quick access or element-specific commands.

This menu can be added in any element with position:relative or position:absolute (read more on positioning)

<ul class="mini-menu">
	<li><a href="#" title="Move down"><img src="images/icons/fugue/arrow-270.png" width="16" height="16"></a></li>
	<li><a href="#" title="Delete"><img src="images/icons/fugue/cross-circle.png" width="16" height="16"> Delete</a></li>
</ul>
