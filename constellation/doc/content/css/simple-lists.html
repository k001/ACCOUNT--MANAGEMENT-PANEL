Arbo

This style requires the file js/list.js to be fully fonctional.

The arbo class is usefull to display arborescent data, with infinite nested levels. You can easily control whether branches are opened or closed (by adding/removing the class closed), use titles and change icons.

To close a branch, add the class closed to it. Note that, for accessibility issues, it is not recommended to apply it in the source-code, but rather at runtime, because if the browser has Javascript disabled or if it stops on an error, the user won't be able to open the branches. Therefore, you should rather use the class close (without d): the template setup function will then automatically close these branches at startup.

Collapsible list

This style requires the file js/list.js to be fully fonctional.

The collapsible list works quite like the arbo class, but with a different styling. It also allows infinite nested levels, collapsing by class adding and icon customization.

To close a branch, add the class closed to it. Note that, for accessibility issues, it is not recommended to apply it in the source-code, but rather at runtime, because if the browser has Javascript disabled or if it stops on an error, the user won't be able to open the branches. Therefore, you should rather use the class close (without d): the template setup function will then automatically close these branches at startup.

<!-- Add the class with-bg to add background -->
<ul class="collapsible-list">
	<li>
	
		<!-- This adds the toggle button -->
		<b class="toggle"></b>
		
		<span>Expanded element</span>
		
		<!-- The class with-icon will add an icon - see simple-list for available icons -->
		<ul class="with-icon icon-group">
		
			<!-- Add the class 'closed' or 'close' (read above) to a branch to collapse it -->
			<li class="close">
			
				<span>Administrators</span>
				
				<!-- There can be infinite nested levels -->
				<ul class="icon-user">
					<li><a href="#">Jane</a></li>
					<li><a href="#">George</a></li>
				</ul>
			</li>
			<li class="close">
				<span>Authors</span>
				<ul class="icon-user">
					<li><a href="#">Erik</a></li>
					<li><a href="#">James</a></li>
					<li><a href="#">Nathan</a></li>
					<li><a href="#">Elwin</a></li>
				</ul>
			</li>
			<li>
				<span>Product managers</span>
				<ul class="icon-user">
					<li><a href="#">John</a></li>
					<li><a href="#">Sam</a></li>
					<li><a href="#">Lena</a></li>
					<li><a href="#">Marc</a></li>

The class no-toggle-icon

When applied to a list, the class no-toggle-icon will hide icons for branches having a toggle element:

<ul class="collapsible-list with-bg">
	<li>
		<b class="toggle"></b>
		<span>Expanded element</span>
		
		<!-- Add the class to any element having 'with-icon' or below -->
		<ul class="with-icon icon-group no-toggle-icon">
			<li class="close">
				
				...
				
			</li>
			<li class="close">
				
				...
				
			</li>
			<li class="close">
				
				...
				
			</li>
			<li class="close">
				
				...
				
		</ul>
	</li>
</ul>
