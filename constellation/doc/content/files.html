Files

The template comes with 2 sets of files: one with bare HTML (usefull if you want to use it with anything else than Php), and one with HTML+Php : it includes some code to handle errors with the build-in error page, combine/compress JS & CSS files, display Php/Apache version, retrieve search results...

The files are organized as below (Php only files are in bold blue):

  • Template files
    • _errors
      • 400.html/php: Custom error 400 page : Bad request (more)
      • 401.html/php : Custom error 401 page : Unauthorized (more)
      • 403.html/php : Custom error 403 page : Forbidden (more)
      • 404.html/php : Custom error 404 page : Page not found (more)
      • 500.html/php : Custom error 500 page : Internal server error (more)
      • 503.html/php : Custom error 503 page : Service Unavailable (more)
      • error.html/php : Final error page with error description script (more)
      • libError.php : A bunch of functions required to handle errors (more)
      • sendReport.html/php : Example page to allow users to send error report (more)
    • css
      • cache : Folder for caching CSS compressed files (more)
      • 960.gs.css : Grid system for fixed layout (more)
      • 960.gs.fluid.css : Grid system for fluid layout (more)
      • block-lists.css : Block lists styles (more)
      • calendars.css : Mini, medium and large calendars (more)
      • common.css : Generic styles for both standard and mobile templates (more)
      • form.css : Form styling (more)
      • gallery.css : Image gallery styles (more)
      • mini.php : Script for combining/compressing CSS files (more)
      • mobile.css : Styles for the mobile template (more)
      • planning.css : Planning grid styles (more)
      • reset.css : CSS reset
      • simple-lists.css : Simple lists styles (more)
      • special-pages.css : Styles for the special pages (more)
      • standard.css : Styles for the standard template
      • table.css : Tables styles (more)
      • wizard.css : Wizard styles (more)
    • images
      • demo : Various images for the demo pages, remove on your website
      • icons
        • browsers : Browsers icons for the 'please update' page (more)
        • finefiles : Icons for the .files style (see credits for more infos)
        • flags : Country flags, used in the demo pages (see credits for more infos)
        • fugue : Some icons from the great Fugue Icon Set (see credits for more infos)
        • web-app : The Web Application Icon Set (see credits for more infos)
        • Some other template icons
      • old-browsers-bg : Images for the older browsers not handling CSS3 gradients
      • old-browsers-screen : Images for the 'please update' page (more)
      • All other template images
    • js
      • cache : folder for caching JS compressed files (more)
      • common.js : Common jQuery functions for both standard and mobile templates (more)
      • html5.js : JS tweak to enable HTML5 support for older browsers (more)

      • jquery.contextMenu.js : Custom context menu plugin (more)
      • jquery.hashchange.js : Library needed to emulate the 'hashChange' event on older browsers, needed for the tabs (see credits for more infos)
      • jquery.tip.js : Tool tips plugin (more)
      • jsmin.php : Library used for combining/compressing CSS files (more)
      • list.js : Template functions for some list styles (more)
      • mini.php : Script for combining/compressing JS files (more)
      • mobile.js : jQuery functions for the mobile template (more)
      • old-browsers.js : older browsers detection (more)
      • searchField.js : ajax search field plugin (more)
      • standard.ie.js : jQuery functions for the standard template (more)
      • standard.js : jQuery functions for the standard template (IE support) (more)
    • mobile
      • index.html/php : Example dashboard page

      • tour-*.html/php : Demo pages with styles use examples
