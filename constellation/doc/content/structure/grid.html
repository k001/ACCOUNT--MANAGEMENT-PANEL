Grid system

960.grid

The template uses a grid system (on the standard template only), the one from 960.gs. It divides the document in 12 columns, and blocks can span accross columns for easyier layout.

It comes in 2 versions:

Fixed
The document is 960px wide, centered in the viewport, and columns are 60px wide. Great if you have little amount of data to display.
Fluid
The document expands to fill the entire viewport, and columns have variable width. Ideal for large sets of data.

To choose which version to use, just comment/uncomment the desired line in the document <head>:

Fixed
<!-- Comment/uncomment one of these files to toggle between fixed and fluid layout -->
<link href="css/960.gs.css" rel="stylesheet" type="text/css">
<!--<link href="css/960.gs.fluid.css" rel="stylesheet" type="text/css">-->
Fluid
<!-- Comment/uncomment one of these files to toggle between fixed and fluid layout -->
<!--<link href="css/960.gs.css" rel="stylesheet" type="text/css">-->
<link href="css/960.gs.fluid.css" rel="stylesheet" type="text/css">

Using the grid

Here is a quick example on how to use the grid system to setup 2 colums side by side:

<article class="container_12">

	<section class="grid_4">
		[Column content goes here]
	</section>
	
	<section class="grid_8">
		[Column content goes here]
	</section>
	
</article>

There are much more classes to use, to learn more on how to use the grid system, go to 960.gs.

Can I use another grid system?

Sure: the template does not rely on any class of the selected grid system, you can use you favorite one.
